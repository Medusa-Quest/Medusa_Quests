# 🧚 Medusa Quest — Компаньон для 100% прохождения Elden Ring

[![Live Demo](https://img.shields.io/badge/🌐-Открыть_сайт-2ea44f?style=for-the-badge)](https://medusa-quest.github.io/Medusa_Quests/)
![GitHub Repo stars](https://img.shields.io/github/stars/Medusa-Quest/Medusa_Quests?style=for-the-badge)

**Medusa Quest** — это интерактивное веб-приложение, созданное одним любителем Элден Ринг, которое поможет вам пройти игру на 100%. Ваш гид — маленький дух Медузки, который следит за вашим прогрессом, подсказывает маршруты и комментирует события.

---

## ✨ Особенности

*   **🗺️ Пошаговое прохождение:** Детальные гайды по каждому акту и локации, от Западного Замогилья до DLC.
*   **📊 Отслеживание прогресса:** Встроенная система XP и уровней Медузки, которая растет вместе с вами.
*   **🧠 Умный чат с ИИ:** Спросите Медузку о боссах или квестах (для работы требуется API-ключ OpenRouter).
*   **🎵 Атмосферная музыка:** Поддерживает фоновое аудио.
*   **📱 Адаптивный дизайн:** Отлично работает на ПК, планшетах и телефонах.

## 🚀 Как использовать

1.  **Онлайн (рекомендуется):** Перейдите по ссылке **[medusa-quest.github.io/Medusa_Quests](https://medusa-quest.github.io/Medusa_Quests/)**. Ничего скачивать не нужно!
2.  **Настройка ИИ:** Чтобы включить чат с Медузкой, нажмите ⚙️ в меню и введите API-ключ от OpenRouter.

## 🛠️ Технологии

*   Чистый HTML5, CSS3 и JavaScript (без фреймворков).
*   Web Audio API для генерации эффектов.
*   Хостинг на GitHub Pages.
